  • 使い方
    1. ログ・ホライズン冒険窓口から「マイページ」→「あなたの作成した〈冒険者〉」から冒険者を選択する。
    2. 「基本情報を変更する」から、「外部ツールからの〈冒険者〉データ参照を許可する」にチェックを入れ、「変更を確定する」を選択する。
    3. 以下の形式のURLを上段のテキストボックスに入力し、「読み込む」を選択する。
      • https://lhrpg.com/lhz/pc?id=XXXXXX
      • https://lhrpg.com/lhz/pc_status?id=XXXXXX
      • https://lhrpg.com/lhz/pc_skill?id=XXXXXX
      • https://lhrpg.com/lhz/pc_item?id=XXXXXX
      • https://lhrpg.com/lhz/sheets/XXXXXX.html
      ※いずれのURLでのこのサイトで動作は変わりません。
      ※反応しない場合、再度「外部ツールからの〈冒険者〉データ参照を許可する」にチェックを入れ、「変更を確定する」を行ってください。
    4. 不具合や要望等はこちらに
  • このサイトでできること
    • ココフォリア用のキャラ駒を出力する。
      右上の ⚙ や各キャラの特技ごとに、出力方法をカスタマイズできる。
      • ※設定はブラウザごとに保存されるため、別のパソコン、スマホ等では共有されません。 ※設定のバックアップは ⚙ で開かれるメニューの最下部にある「バックアップの作成」から、復元は「バックアップから復元」から行えます。 (バックアップから復元した場合、ページの再読み込みが行われます。) ※特技ごとのチャパレ設定はキャラクターごとに個別です。
    • エネミーデータの駒作成（実装予定）
    • ダメージ計算機（実装予定）
